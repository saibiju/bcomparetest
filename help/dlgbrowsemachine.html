Selecting a Remote Registry

If the remote computer you want to connect with does not appear in the list, you may need to do the following:

Turn on network discovery and file sharing

1.In Windows Control Panel, pick Network and Internet > Network and Sharing Center > Advanced sharing settings.
2.Pick Turn on network discovery and Turn on file and printer sharing.
3.Click Save changes.

Enable the Remote Registry service

1.In Windows Computer Management, pick Services and Applications > Services.
2.In the services list, double-click on Remote Registry.
3.Click Start and wait for the status to update.
4.Change the Startup type to Automatic.
5.Click OK.

Allow connections through Windows Firewall

1.In Windows Control Panel, pick System and Security > Windows Firewall >Allowed Programs.
2.In the allowed programs list, check Windows Remote Management.
3.Click OK.
