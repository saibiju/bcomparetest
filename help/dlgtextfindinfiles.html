Finding Text in Files

Search > Find in Files

To locate specific text in multiple files, pick Search > Find in Files .

Text to find

Enter the text to search for.

Folder

Enter the name of the folder to search in.

File mask

Enter the mask of files to include in the search.

Match character case

Match only text with the same uppercase and lowercase characters.

Whole words only

Match on the whole words but not parts of words.

Regular expression

Indicates the search string is a PCRE regular expression.  Click on the Insert Regular Expression Element button to pick from a list of common regular expression terms.  Click the RE Examples button for more info.

Search subfolders

Recursively searches folders within the main one.
